구체성

css의 선택자 구체성 0,0,0,0 이면 좌측이 크면 클수록 높은 구체성을 가진다.

  1. 0, 1, 0, 0: 선택자에 있는 모든 id속성값
  2. 0, 0, 1, 0: 선택자에 있는 모든 class 속성값, 기타 속성, 가상 클래스
  3. 0, 0, 0, 1: 선택자에 있는 모든 요소, 가상요소
  4. 전체 선택자는 0,0,0,0을 가진다.
  5. 조합자는 구체성에 영향을 주지 않는다.(>,+)

예시

h1{} => 0,0,0,1
.a{} => 0,0,1,0
#page{} => 0,1,0,0
div#page{} => 0,1,0,1
